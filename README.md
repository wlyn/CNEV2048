# 🚗 新能源汽车 2048

一款以中国新能源汽车品牌为主题的 2048 游戏！

## 🎮 在线体验

👉 **[点击这里开始游戏](https://你的用户名.github.io/ev-2048/)**

> 部署后请将上方链接中的 `你的用户名` 替换为你的 GitHub 用户名

## 🎯 游戏规则

- 使用 **方向键** 或 **WASD** 移动所有方块
- 相同品牌的方块相撞会**合并升级**
- 目标：合并到 **比亚迪（2048）** 即为胜利！

## 📊 品牌进化图谱

```
极氪(2) → 小米(4) → 零跑(8) → 蔚来(16) → 小鹏(32) → 理想(64)
    ↓
  理想(64) → 埃安(128) → 五菱(256) → 长安(512) → 吉利(1024) → 比亚迪(2048)
```
